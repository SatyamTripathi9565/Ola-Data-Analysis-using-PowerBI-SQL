# OLA Ride Booking Data Analysis

## Project Overview
This project analyzes OLA ride booking data using SQL and Power BI to uncover insights related to ride volume, cancellations, revenue, customer behavior, and vehicle performance.

## Tools & Technologies
- Power BI
- SQL

## Key Features
- Ride Volume Analysis
- Booking Status Breakdown
- Revenue Analysis
- Cancellation Insights
- Customer vs Driver Ratings
- Vehicle Type Performance

## SQL Analysis
Implemented SQL queries for:
- Successful bookings
- Average ride distance
- Customer cancellation analysis
- Top customers
- Revenue calculation
- Rating analysis

## Dashboard Insights
- Peak booking periods
- Revenue trends
- Most preferred vehicle types
- Cancellation reasons
- Payment method analysis

## Files Included
- Power BI Dashboard (.pbix)
- SQL Queries
- Dataset
- Dashboard Screenshots
